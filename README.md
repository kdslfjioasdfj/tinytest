# TinyTest

**TinyTest** is a minimal, header-and-source-based unit testing library for C.  
It is cross-platform, works on C90/C89 compilers, and is designed to be simple, small, and easy to integrate into any project.

---

## Features

- Single header + source file (`tinytest.h` + `tinytest.c`)
- Cross-platform: Linux, macOS, Windows
- Builds as **static** or **shared** library
- No dependencies, minimal setup
- Simple API with `tinytest_expect` and `tinytest_assert`

---

## Getting Started

1. Installing **TinyTest**

```bash
git clone https://github.com/kdslfjioasdfj/tinytest
cd tinytest
```

2. Building **TinyTest**

```bash
cmake -B build
cmake --build build
```

3. Reading the documentation
   Documentation is at [docs.md](./docs.md)
